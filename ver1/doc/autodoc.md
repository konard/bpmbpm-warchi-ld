## auto documentation requirements 
### general requirements
Алгоритм обновления документации в папке doc, т.е. https://github.com/bpmbpm/warchi-ld/blob/main/ver1/doc/ (авто документирование проекта):
- обновляй автоматически файлы с описанием кода https://github.com/bpmbpm/warchi-ld/blob/main/ver1/doc/. Файлы папки Custom требует отдельного (явного) указания на обновление.
- при обновлении вначале каждого файла указывай Pull Request по которому происходило его обновление и дату обновления
- при описании функции - указывай модуль
- язык - русский, схемы в формате mermaid
### file requirements
- создавай (обновляй) файл Folder_Structure.md. Он должен содержать дерево папок \ файлов и их размер. При формировании doc/Folder_Structure.md при числе строк в файле с кодом более 100 строк делай округление двух последних разрядов, например, 1165 округляй до 1200.
